Asset lists for the Cosmos ‚öõÔ∏è

```
npm install @chain-registry/assets
```

```js
import { asset_lists } from '@chain-registry/assets';
const assetList = asset_lists
    .find(asset => asset.chain_name === 'osmosis');
```
